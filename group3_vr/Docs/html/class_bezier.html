Detailed Description

Generic class which is used to draw Bezier curves between an origin and Destinations.

This code draws a cubic bezier curve, starting at the origin and finishing at the destination The height of the line is based on the distance between the lines. Code based on: https://gist.github.com/valryon/b782bb76f543354f11f7
